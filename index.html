🔥 Connect Firebase

To make VOXLO work with real users, paste your Firebase config below:

How to get your config:
1. Go to console.firebase.google.com
2. Create project → name it voxlo
3. Project Settings ⚙️ → Your Apps → Web → Register
4. Copy the firebaseConfig object and paste above

Also enable in Firebase console:
• Authentication → Email/Password
• Firestore Database → Create (test mode)
• Storage → Get started
